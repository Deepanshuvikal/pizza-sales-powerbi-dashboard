# 🍕 Pizza Sales Dashboard (Power BI)

## 📊 Project Overview

This project is a Pizza Sales Analysis Dashboard built using Power BI.
It provides insights into sales performance, customer behavior, and product trends.

## 🔧 Tools Used

* Power BI
* Power Query
* DAX

## 📈 Key Insights

* Total Revenue, Orders, and Quantity
* Sales trends (daily & monthly)
* Top & bottom performing pizzas
* Category & size analysis
